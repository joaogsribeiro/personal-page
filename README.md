# Portfólio Pessoal - Atividade Prática 2

Projeto desenvolvido para a disciplina GAC116 - Programação Web - 2026/1, correspondente à Atividade Prática 2 - Desenvolvimento de Página Pessoal. 

O objetivo central é aplicar conceitos de estruturação de páginas, estilização, responsividade e organização de código, utilizando dois frameworks CSS distintos para fins de comparação.

## 🌐 Acesso ao Projeto

**Acesse a versão em produção (GitHub Pages):**

- [Versão Bootstrap](https://joaogsribeiro.github.io/personal-page/versao_bootstrap/index.html)
- [Versão Bulma](https://joaogsribeiro.github.io/personal-page/versao_bulma/index.html)

## ⚙️ Tecnologias e Frameworks Utilizados

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização via frameworks.
* **JavaScript (Vanilla):** Manipulação do DOM para a lógica de alternância de temas.
* **Bootstrap 5.3:** Framework CSS primário (Versão 1).
* **Bulma 1.0:** Framework CSS alternativo (Versão 2) focado em flexbox e classes semânticas.

## 📋 Requisitos Atendidos

- [x] Desenvolvimento em HTML, CSS e JavaScript.
- [x] Inclusão de informações reais (nome, formação, projetos, contato) baseadas em meu currículo profissional.
- [x] Layout 100% responsivo em ambas as versões.
- [x] Duas versões distintas (Bootstrap e Bulma).
- [x] Funcionalidade de tema Claro/Escuro (Dark Mode) com *toggle* via JavaScript.
